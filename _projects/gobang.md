## Overview

A simple online GoBang project developed using Qt for the client and Python for the server. The AI implementation uses minimax algorithm with α-β pruning and heuristic functions for efficiency optimization.

## Features

- **Player vs AI**: Challenge the AI opponent
- **Player vs Player**: Online multiplayer support (Python server)
- **AI Algorithm**: Minimax with α-β pruning, heuristic functions, and C++ parallel computing
- **Cross-platform GUI**: Built with Qt (C++)

## Tech Stack

- **Qt** — Cross-platform GUI framework (C++)
- **Python** — Backend server
- **Minimax + Alpha-Beta Pruning** — AI algorithm
