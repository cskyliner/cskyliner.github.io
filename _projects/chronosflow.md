## Overview

ChronosFlow 是一个基于 PySide6 的跨平台时间管理应用，作为北京大学 2025 年春季《程序设计实习》课程大作业。项目专注于可视化日程管理，结合直观交互与美观设计，支持桌面通知提醒与托盘化，帮助 PKUer 实现更高效的时间规划。

## Features

- **日历主视图**：交互式月视图，支持点击查看与编辑事件
- **年度热力图**：GitHub 配色风格展示全年任务密度
- **周视图日程**：按小时粒度展示每周任务分布
- **Upcoming**：展示近期 DDL，支持增加、修改与删除
- **深浅主题自动适配**：自动识别系统外观
- **SQLite 数据库**：本地存储事件数据
- **AI 日程规划助手**：集成 DeepSeek 大模型 API

## Tech Stack

- **PySide6** — Qt 跨平台 GUI 框架
- **Python 3.10+** — 开发语言
- **SQLite** — 数据存储
- **DeepSeek API** — AI 智能日程规划
